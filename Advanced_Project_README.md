# Heart Disease Prediction

## Description
The objective of this project is to predict whether a person has heart disease using machine learning. Early detection can help in taking preventive measures and improving health outcomes.

## Dataset
- Source: Kaggle  
- Description of data: 
  The dataset contains medical information such as age, sex, blood pressure, cholesterol levels, heart rate, and other health-related features.  
  The target variable indicates:
  - 0 = No heart disease  
  - 1 = Heart disease  

## Steps Performed

 1. Data Cleaning
- Checked for missing values  
- Ensured correct data types  

2. Exploratory Data Analysis
- Studied feature distributions  
- Analyzed relationships between variables  
 
3. Model Building
- Split data into training and testing sets  
- Normalized data using StandardScaler  
- Trained model using Random Forest Classifier  

## Results
- The model predicts heart disease with good accuracy  
- Metrics used:
  - Accuracy  
  - Precision  
  - Recall  
  - F1 Score  
  - ROC-AUC Score  

## Tools Used
- Python  
- NumPy  
- Pandas  
- Scikit-learn  

## Conclusion
This project demonstrates how machine learning can be used in healthcare to predict heart disease. It helps in early detection and supports better decision-making.
