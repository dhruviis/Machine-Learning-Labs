# Spam Email Detection

## Description
This project is used to detect whether an email is spam or not using Python. It helps in identifying unwanted or harmful emails.

## Dataset
- Source: Kaggle / CSV file
- Description of data:
The dataset contains email text with labels like "spam" and "not spam".

## Steps Performed
1. Data Cleaning  
   - Cleaned the email text by removing unnecessary words and symbols  

2. Exploratory Data Analysis  
   - Checked how many spam and non-spam emails are present  

3. Visualization  
   - Used simple graphs to understand the data  

4. Model Building  
   - Used a machine learning model to classify emails  

## Results
- The model can predict if an email is spam or not  
- Got good accuracy  

## Tools Used
- Python  
- Libraries: NumPy, Pandas, Matplotlib, Scikit-learn  

## Conclusion
This project shows how machine learning can help in detecting spam emails and keeping inboxes clean.
